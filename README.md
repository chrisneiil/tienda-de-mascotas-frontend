# Tienda de Mascotas FrontEnd

Proyecto desarrollado para la actividad sumativa de la asignatura Desarrollo Full Stack II - DSY2202.

## Descripción

La aplicación corresponde a una tienda online de productos para mascotas. Permite visualizar productos, simular compras, registrar usuarios y administrar productos y usuarios.

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Git y GitHub

## Roles del sistema

### Cliente
- Iniciar sesión
- Registrarse
- Recuperar contraseña
- Modificar perfil
- Ver catálogo
- Agregar productos al carrito
- Simular pago

### Administrador
- Administrar productos
- Administrar usuarios
- Revisar información general del sistema

## Páginas desarrolladas

- Inicio
- Inicio de sesión
- Registro de usuario
- Recuperar contraseña
- Modificar perfil
- Catálogo de productos
- Carrito de compras
- Pago exitoso
- Administración de productos
- Administración de usuarios

## Validaciones implementadas

- Campos obligatorios
- Validación de correo electrónico
- Validación de contraseña entre 8 y 16 caracteres
- Contraseña con mayúscula
- Contraseña con número
- Contraseña con carácter especial
- Confirmación de contraseña
- Validación de teléfono
- Validación de precio y stock

## Responsividad

La aplicación utiliza Bootstrap y el sistema GRID de 12 columnas para adaptarse a:

- Dispositivos móviles
- Tablets
- Computadores de escritorio
